This is the presentation material presented on the 11th of June 2019 for the Hannover Rust meetup.

The presentation can be served by Rocket and can be run compiling / running the project with 0
`cargo run --release` and then pointing your browser to `localhost:8000/slides/index.html`

The example code to the presentation is located in the `/examples/syntax` project.

The example code to questions is located in the `/examples/questions` project.
